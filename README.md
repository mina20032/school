# منصة الاختبارات التعليمية

منصة ويب كاملة للاختبارات التعليمية مبنية بـ **Python و Flask**، واجهة الموقع بالكامل **بالعربية**، وتدعم ثلاثة أدوار: **طالب**، **مشرف (معلم)**، **مشرف أعلى**.

## التقنيات

- **الخادم:** Python مع Flask
- **قاعدة البيانات:** SQLite (افتراضي) أو PostgreSQL عبر `DATABASE_URL`
- **الواجهة:** HTML، CSS، JavaScript
- **التصميم:** متجاوب (موبايل وسطح المكتب)
- **الاتجاه:** من اليمين لليسار (RTL) للعربية

## هيكل المشروع

```
quiz_platform/
├── app.py           # تطبيق Flask والمسارات
├── config.py       # الإعدادات (قاعدة البيانات، الصفوف بالعربية)
├── models.py       # نماذج SQLAlchemy
├── auth.py         # التحقق: student_required, admin_required, super_admin_required
├── requirements.txt
├── README.md
├── templates/      # قوالب HTML (عربي)
└── static/
    ├── css/style.css
    └── js/main.js, quiz.js
```

## تشغيل المشروع

### 1) إنشاء بيئة افتراضية (مُفضّل)

```bash
cd quiz_platform
python -m venv venv
```

- **Windows:** `venv\Scripts\activate`
- **Linux/macOS:** `source venv/bin/activate`

### 2) تثبيت المتطلبات

```bash
pip install -r requirements.txt
```

### 3) تشغيل التطبيق

```bash
python app.py
```

- العنوان: **http://127.0.0.1:5000/**

### إصلاح أخطاء الـ IDE (Unresolved reference)

إذا ظهرت أخطاء مثل `Unresolved reference 'flask_login'` في المحرر:

1. تأكد أنك فتحت المجلد **School** (وليس pythonProject) كمشروع في Cursor/VS Code.
2. اختر مفسر بايثون البيئة الافتراضية: **Ctrl+Shift+P** → "Python: Select Interpreter" → اختر `quiz_platform\venv\Scripts\python.exe`.
3. أو شغّل من داخل المجلد: `quiz_platform\setup_venv.bat` لإنشاء الـ venv وتثبيت الحزم، ثم اختر المفسر كما في 2.
- عند أول تشغيل يتم إنشاء قاعدة البيانات وحساب **مشرف أعلى** افتراضي.

### 4) تسجيل الدخول للمشرف الأعلى (أول مرة)

- **المسار:** اكتب في المتصفح: `http://127.0.0.1:5000/admin`
- **اسم المستخدم:** `super`
- **كلمة المرور:** `super123`

يُنصح بتغيير كلمة المرور من لوحة المشرف الأعلى → إدارة المستخدمين → تغيير كلمة المرور.

## الأدوار والصلاحيات

| الدور | الوصف |
|-------|--------|
| **طالب** | لا يحتاج حساب. يدخل الاسم والصف (أولى / ثانية / ثالثة إعدادي) ثم يحل الاختبارات. لا يرى رابط لوحة الإدارة. |
| **مشرف (معلم)** | تسجيل دخول من `/admin` فقط. إضافة مواد وأسئلة، تعديل وحذف، عرض النتائج ومحاولات الطلاب. |
| **مشرف أعلى** | نفس المشرف + عرض كل الطلاب والنتائج والمستخدمين، إنشاء/حذف مشرفين، تغيير كلمات المرور، سجل النشاط. |

---

## الطالب

- إدخال **الاسم** واختيار **الصف:** أولى إعدادي، ثانية إعدادي، ثالثة إعدادي.
- عرض المواد المتاحة لصفه.
- كل اختبار يُحل **مرة واحدة فقط** (منع إعادة الحل).
- أسئلة **اختيار من متعدد** (4 خيارات)، مع تحديد الإجابة الصحيحة ونقاط لكل سؤال.
- عند التسليم: تصحيح تلقائي، احتساب النقاط، حفظ النتيجة وعنوان IP ووقت التسليم.
- **لوحة الطالب:** اسم الطالب، الصف، مجموع النقاط، عدد الأسئلة الصحيحة، المواد التي قام بحلها، سجل الاختبارات.
- **ماراثون الطلاب:** ترتيب حسب مجموع النقاط مع أيقونات 🥇 الأول، 🥈 الثاني، 🥉 الثالث.

---

## منع الغش

- **مؤقت للاختبار:** يظهر وقت متبٍ ويُسلّم تلقائياً عند انتهائه.
- **منع إرسال مكرر:** جلسة اختبار واحدة لكل محاولة.
- **منع إعادة حل نفس الاختبار:** بعد التسليم لا يمكن حل نفس المادة مرة أخرى.
- **حفظ وقت التسليم وعنوان IP** في جدول الإجابات.

## قاعدة البيانات

- **students:** id, name, grade, total_points, created_at
- **subjects:** id, name, grade
- **questions:** id, subject_id, question_text, option_a/b/c/d, correct_answer, points
- **answers:** id, student_id, question_id, selected_answer, is_correct, points_awarded, submitted_at, ip_address
- **users:** id, username, password, role (admin | super_admin)
- **logs:** id, action, timestamp
